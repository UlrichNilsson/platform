# Bootstrap lab

## Install and run

* `npm install`
* `npm install bootstrap@4.0.0 --save`
* `npm install jquery@3.3.1 popper.js@1.12.9 --save`
* `npm start`

## Later

* `npm install font-awesome@4.7.0 --save`
* `npm install bootstrap-social@5.1.1 --save`
